# Client Notes

## Raw Concept Notes
Client requested a system where the hydraulic cylinder is attached to a spindle gear to create rotation.

At the end of the spindle gear, add three smaller cylinders installed in a circular beveled housing. These smaller cylinders are intended to slowly grip the bottle top while it is being pressed down. The assembly should help guide the top onto the bottle as it rotates.

Include a manifold and adjustable pressure reducing valves so the bottle does not break during the torque process.

Use 4-20 mA pressure transducers and a load cell, wired to a programmable logic controller, so torque and pressure can be monitored and adjusted.

Add a human machine interface so process parameters can be changed as needed.

Client also mentioned future interest in I/O connected to a server, internet-based operating features, and cameras for remote monitoring.

## Cleaned Summary
The client wants a bottle topping system that:
- rotates a spindle using hydraulic actuation
- presses and rotates the bottle top at the same time
- uses a 3-point hydraulic gripping mechanism to guide and hold the top
- limits force using pressure reducing valves
- measures pressure and load
- monitors or estimates torque
- uses a PLC for control and an HMI for operator settings
- may later include remote monitoring functions
