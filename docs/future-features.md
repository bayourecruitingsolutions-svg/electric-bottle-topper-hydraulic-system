# Future Features

## Purpose
This document captures ideas that are interesting but not yet part of the base project scope.

## Possible Future Features
- Remote monitoring dashboard
- Camera integration for remote viewing
- Data logging to local or cloud server
- Recipe management for multiple bottle/cap types
- Production reporting
- Trend charts for torque and pressure
- User account roles and permissions
- Barcode or product identification
- Predictive maintenance alerts
- Expanded diagnostics screens
- Network-connected I/O
- Remote notification of alarms

## Notes on Scope
These features should be treated as optional until the base machine concept is defined and documented.

## Current Recommendation
Focus first on:
- base machine concept
- local control
- safety
- pressure management
- torque monitoring
- operator interface

Then revisit future features later as a separate phase.
