# Assumptions

## Purpose
This document records current assumptions being made during the planning phase of the Electric Bottle Topper project.

## Current Assumptions
- The machine will use hydraulic actuation for both rotational drive and gripping functions.
- A primary hydraulic cylinder will drive a spindle gear to create rotation.
- Three smaller cylinders will be used to grip and guide the bottle top.
- Pressure reducing valves will be required to avoid damaging glass bottles.
- A PLC will be the primary machine controller.
- A local HMI will be used for operator interaction.
- Pressure transducers will use 4-20 mA signals.
- A load cell will be used to help estimate or monitor torque.
- Remote monitoring may be considered in the future, but is not required for the initial documentation phase.
- The current repository is for planning and note capture, not final design approval.

## Assumptions to Validate Later
- Bottle sizes and top/cap types are still unknown.
- Required torque range is still unknown.
- Required cycle time is still unknown.
- Required production volume is still unknown.
- Final PLC and HMI platform are not yet selected.
- Final hydraulic pressure ranges are not yet known.
- Mechanical geometry for torque calculation is not yet defined.
