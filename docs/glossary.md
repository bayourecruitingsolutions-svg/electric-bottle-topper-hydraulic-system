# Project Glossary

- **HMI (Human Machine Interface):** The touchscreen or physical control panel where the operator interacts with the machine, views data, and changes settings.
- **PLC (Programmable Logic Controller):** The industrial computer that reads the sensors, runs the logic, and controls the hydraulic valves and motors.
- **PRV (Pressure Reducing Valve):** A mechanical hydraulic valve used to lower the pressure in a specific part of the circuit (used here to prevent glass breakage).
- **Load Cell:** An electronic sensor that measures applied force or weight.
- **Differential Pressure:** The difference in pressure between the supply line and the return line. Useful for understanding how much actual work/force the cylinder is exerting.
- **Spindle Gear:** The mechanical rotating component that interfaces with the hydraulic cylinder to turn linear motion into the rotational motion needed to screw on the bottle top.
- **4-20 mA:** An analog electrical standard used for industrial sensors. 4 milliamps represents the lowest reading (e.g., 0 PSI), and 20 milliamps represents the highest reading (e.g., 3000 PSI).
