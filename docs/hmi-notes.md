# HMI Notes

## Purpose
This document captures ideas for the Human Machine Interface (HMI) screens, displays, and operator controls.

## Main Screen Ideas
- Machine status
- Current mode (Auto / Manual / Fault / E-stop)
- Pump status
- Supply pressure
- Return pressure
- Differential pressure
- Load cell reading
- Calculated torque
- Start / Stop / Reset buttons
- Alarm banner

## Settings Screen Ideas
- Pressure setpoint
- Torque limit
- Cycle timer
- Extend time
- Retract time
- Grip pressure setting
- Recipe selection
- Operator access levels

## Alarm Screen Ideas
- Active alarms
- Alarm history
- Time/date stamp
- Alarm reset button
- Fault descriptions

## Maintenance / Diagnostics Screen
- Live I/O status
- Analog values raw and scaled
- Valve output status
- Sensor health indicators
- Calibration access
- Manual jog controls

## User Access Thoughts
- Operator level
- Maintenance level
- Engineering/admin level

## Notes
- Final HMI layout should be simple and readable.
- Critical values should be easy to see during operation.
- Parameter edits should be limited by authorized access levels.
