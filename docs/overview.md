# Project Overview

## Name
Electric Bottle Topper

## Overview
Electric Bottle Topper is a concept for an electro-hydraulic machine that presses and rotates a bottle top onto a bottle using controlled hydraulic motion, spindle rotation, pressure regulation, and sensor-based monitoring.

## Intended Functions
- apply downward force to a bottle top
- rotate the bottle top during application
- guide and grip the bottle top during the process
- monitor pressure and load
- estimate torque
- stop operation under fault or emergency conditions

## Main Subsystems
- hydraulic power unit
- hydraulic cylinder for spindle drive
- spindle gear and rotating head
- three-cylinder gripping assembly
- pressure sensing
- load sensing
- PLC
- HMI
- safety shutdown and E-stop functions

## Current Phase
The project is currently in concept documentation. This repository is being used to collect notes and organize ideas before detailed engineering begins.
