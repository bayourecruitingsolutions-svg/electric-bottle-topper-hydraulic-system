## [Date: 2026-03-28] - Initial Client Request
**Attendees:** Client, Engineering
**Topic:** Initial Project Scope

**Key Takeaways:**
- Client wants an electro-hydraulic bottle topping machine.
- Needs a main hydraulic cylinder attached to a spindle gear for rotation.
- Needs a 3-cylinder gripping head in a beveled housing to guide the bottle top.
- Must include pressure reducing valves to prevent breaking the glass.
- Wants to monitor torque/pressure using a load cell and 4-20mA pressure transducers.
- Needs a PLC for control and an HMI for parameter adjustment.
- Client mentioned future interest in an internet-based OS, remote server I/O, and cameras to potentially sell to larger companies (e.g., Coca-Cola).
