# Bill of Materials (BOM) - Draft

This is a preliminary list of major components needed for the Electric Bottle Topper. Part numbers and exact specs will be added during the design phase.

## Hydraulic Components
- [ ] Hydraulic Power Unit (Pump, Motor, Reservoir)
- [ ] Primary Hydraulic Cylinder (for spindle drive)
- [ ] 3x Small Hydraulic Cylinders (for gripping head)
- [ ] Hydraulic Manifold
- [ ] Directional Control Valves (Forward/Reverse electrically actuated)
- [ ] Adjustable Pressure Reducing Valves (PRVs)
- [ ] Pressure Relief Valve (Safety)

## Electrical & Control
- [ ] Programmable Logic Controller (PLC)
- [ ] Human Machine Interface (HMI) panel
- [ ] Motor Starter / Contactor
- [ ] 24VDC Power Supply
- [ ] Emergency Stop Pushbutton & Safety Relay
- [ ] I/O blocks (Local and potentially remote)

## Instrumentation & Sensors
- [ ] Supply Pressure Transducer (4-20 mA)
- [ ] Return Pressure Transducer (4-20 mA)
- [ ] Load Cell
- [ ] Load Cell Amplifier / Signal Conditioner (compatible with PLC)
- [ ] High-Pressure Limit Switch (Hardware safety)
- [ ] Low-Pressure Limit Switch

## Mechanical Hardware
- [ ] Spindle Gear Assembly
- [ ] Circular Beveled Housing (for gripping head)
- [ ] Machine Frame / Guarding
